# campusTAKESHI

キャンパスライフサポーターたけし
「赤ペン先生たけし(仮)」

EVP
「赤ペン先生たけしは、授業の内容の細かい内容を確認したいという問題を解決するサービスです。
シラバスを中心に履修登録を組んでる筑波大生向けの、生徒目線の授業の評価を見られるサービスです。
これは、シラバスに課題や授業の大変さなどの生徒目線の追加情報を提供することで、ただシラバスを見るだけでなく、生徒側の意見を踏まえた授業選びが可能になります」
